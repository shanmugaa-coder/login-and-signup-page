# login-and-signup-page
A responsive login page built using HTML and CSS, featuring a clean UI design, structured layout, and modern styling for an improved user experience..
🍔 Canteen Food Order System
A modern and visually appealing login page for a Canteen Food Order System, built using HTML and CSS.

 ✨ Features
- 🔐 Student Login Form (Name, Email, Password)
- 🎨 Glassmorphism UI Design
- 🌄 Full-screen background image with overlay
- ⚡ Smooth animations (fade-in effect)
- 📱 Clean and structured layout
- 🎯 User-friendly interface

 🛠️ Technologies Used
- HTML5
- CSS3 (Flexbox, Animations, Backdrop Filter)

 📌 Project Overview
This project is a front-end design for a canteen food ordering system.  
It allows students to log in using their credentials and provides a clean and attractive interface.

The design focuses on:
- Modern UI (Glass effect)
- Better user experience
- Proper alignment and spacing
